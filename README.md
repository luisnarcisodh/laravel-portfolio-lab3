# Laravel Portfolio Website

## Project Title
Laravel Portfolio Website

## Project Description
This project is a simple multi-page portfolio website created using the Laravel framework.
It was developed as part of Laboratory 2 to practice Laravel installation, routing, controllers,
and Blade templates.

The project contains three pages: a main portfolio page and two sub-pages.
Each page is handled by its own controller and rendered using Blade templates.

## Pages Included
- Main Portfolio Page
- About Me Page
- Hobbies Page

## How to Run the Project Locally
1. Clone the repository to your local machine  
2. Open the project folder  
3. Run `composer install`  
4. Copy `.env.example` and rename it to `.env`  
5. Run `php artisan key:generate`  
6. Run `php artisan serve`  
7. Open your browser and go to `http://127.0.0.1:8000`
